The Basics of the Web & HTML

Overview

The World Wide Web is a collection of billions of web pages that can be accessed through the internet. Web pages are documents written in Hyper Text Markup Language (HTML) that are stored on servers - which are just computers that are specially designed for hosting web pages.

To access a web page a person will need a computer with a web browser and a connection to the internet. The web browser will make a request through the internet to a server by using Hypertext Transfer Protocol (HTTP). The server will send the file to the user's computer, and the web browser will interpret the HTML and display the web page.

HTML

An HTML document will provide the structure and content of a web page by using different elements. Elements are made up of tags, content, and attributes. Most elements will normally have a start tag and an end tag, which will define the element type. In between the tags will be the text content that will be displayed on the web page. Elements can also contain attributes, which contain additional information about the element.

Some elements do not have any content, so rather than having a start and closing tag there is just a single tag. These are called void tags. One example of a void tag is for a line break.

Also, some types of tags are considered container tags. There are two sub-types of container tags: inline tags, which refer to a specific segment of text content, and block tags, which are like an invisible box around a block of content.

There are many different types of elements, so rather than memorizing them all it is best to understand how they work, and then if you need a new element you can search the internet.
